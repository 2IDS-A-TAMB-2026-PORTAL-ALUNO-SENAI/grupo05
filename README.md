# 🎓 Portal do Aluno SENAI

## 🇧🇷 Descrição do Projeto

Este projeto consiste no desenvolvimento da página inicial do **Portal do Aluno SENAI**, com foco em acessibilidade, responsividade e experiência do usuário.

A aplicação simula um sistema de cadastro de alunos para cursos do SENAI, utilizando tecnologias modernas de desenvolvimento web e recursos de acessibilidade digital.

---

## 🇧🇷 Objetivo

Desenvolver uma página web completa utilizando:

* HTML5
* CSS3
* JavaScript
* Bootstrap 5
* SweetAlert2

O sistema deve apresentar uma interface moderna, responsiva e acessível, garantindo uma navegação intuitiva para todos os usuários.

---

## 🇧🇷 Requisitos Implementados

### Cabeçalho

* Exibição do nome ou logo "Portal do Aluno SENAI";
* Menu de navegação responsivo utilizando Bootstrap;
* Botões de acessibilidade visíveis e funcionais.

### Banner Principal

* Título de destaque;
* Texto explicativo sobre o portal;
* Botão "Realizar Cadastro";
* Layout adaptável a diferentes tamanhos de tela.

### Formulário de Cadastro

O formulário contém os seguintes campos:

* Nome Completo;
* E-mail;
* Telefone;
* Curso Desejado (Select);
* Turno (Radio Button);
* Aceite dos Termos (Checkbox).

### Validações

* Verificação dos campos obrigatórios;
* Validação do aceite dos termos;
* Feedback ao usuário em caso de erro ou sucesso.

### Integração com SweetAlert2

* Exibição de mensagem de sucesso ao realizar o cadastro corretamente;
* Exibição de mensagem de erro quando houver campos obrigatórios não preenchidos.

### Recursos de Acessibilidade

* Integração do plugin VLibras para tradução em Libras;
* Botão de Alto Contraste para ativar e desativar o modo de contraste;
* Botão para aumentar o tamanho da fonte;
* Botão para diminuir o tamanho da fonte;
* Utilização de HTML semântico;
* Foco acessível nos elementos interativos;
* Descrições adequadas em botões e campos do formulário.

### Responsividade

O layout foi desenvolvido para funcionar corretamente em:

* Computadores;
* Tablets;
* Smartphones.

Para isso, foram utilizados recursos do Bootstrap 5, como:

* Containers;
* Grid System;
* Cards;
* Alinhamentos;
* Espaçamentos responsivos.

---

## 🇧🇷 Tecnologias Utilizadas

* HTML5
* CSS3
* JavaScript
* Bootstrap 5 (CDN)
* SweetAlert2 (CDN)
* VLibras

---

## 🇧🇷 Resultados Esperados

O projeto busca atender aos seguintes objetivos:

* Página visualmente organizada e responsiva;
* Utilização correta do Bootstrap;
* Integração funcional do SweetAlert2;
* Recursos reais de acessibilidade implementados;
* Formulário validado corretamente;
* Estrutura HTML semântica;
* Experiência do usuário clara e intuitiva;
* Código organizado e funcional.

# 🎓 SENAI Student Portal

## 🇺🇸 Project Description

This project consists of developing the homepage of the **SENAI Student Portal**, focusing on accessibility, responsiveness, and user experience.

The application simulates a student registration system for SENAI courses, using modern web development technologies and digital accessibility resources.

---

## 🇺🇸 Objective

Develop a complete web page using:

* HTML5
* CSS3
* JavaScript
* Bootstrap 5
* SweetAlert2

The system must provide a modern, responsive, and accessible interface, ensuring intuitive navigation for all users.

---

## 🇺🇸 Implemented Requirements

### Header

* Display of the name or logo "SENAI Student Portal";
* Responsive navigation menu using Bootstrap;
* Visible and functional accessibility buttons.

### Main Banner

* Highlighted title;
* Informative text about the portal;
* "Register Now" button;
* Layout adaptable to different screen sizes.

### Registration Form

The form includes the following fields:

* Full Name;
* Email;
* Phone Number;
* Desired Course (Select);
* Study Shift (Radio Button);
* Terms Acceptance (Checkbox).

### Validations

* Verification of required fields;
* Validation of terms acceptance;
* User feedback in case of success or error.

### SweetAlert2 Integration

* Success message displayed when registration is completed correctly;
* Error message displayed when required fields are missing.

### Accessibility Features

* VLibras plugin integration for Brazilian Sign Language translation;
* High Contrast button to enable and disable contrast mode;
* Increase Font Size button;
* Decrease Font Size button;
* Proper use of semantic HTML;
* Accessible focus on interactive elements;
* Appropriate descriptions for buttons and form fields.

### Responsiveness

The layout was developed to work correctly on:

* Desktop computers;
* Tablets;
* Smartphones.

To achieve this, Bootstrap 5 resources were used, including:

* Containers;
* Grid System;
* Cards;
* Alignments;
* Responsive spacing.

---

## 🇺🇸 Technologies Used

* HTML5
* CSS3
* JavaScript
* Bootstrap 5 (CDN)
* SweetAlert2 (CDN)
* VLibras

---

## 🇺🇸 Expected Results

The project aims to achieve the following objectives:

* Visually organized and responsive page;
* Correct use of Bootstrap;
* Functional SweetAlert2 integration;
* Real accessibility features implemented;
* Properly validated form;
* Semantic HTML structure;
* Clear and intuitive user experience;
* Organized and functional code.
